# F1 2025 Emilia-Romagna GP Predictions

Data-driven prediction engine leveraging historical F1 race data, feature engineering, and machine learning to model race performance and finishing positions.

## Overview
This project builds a machine learning pipeline to predict Formula 1 race outcomes by analyzing driver performance, constructor strength, grid position impact, and circuit-specific trends. It applies data preprocessing, exploratory analysis, and predictive modeling to simulate real-world race dynamics.

## Key Features
- Data cleaning and preprocessing of historical F1 datasets  
- Exploratory Data Analysis (EDA) on drivers, teams, and circuits  
- Feature engineering (driver form, grid impact, consistency metrics)  
- Machine learning models for race outcome prediction  
- Model evaluation and performance comparison  

## Tech Stack
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  
- Google Colab
